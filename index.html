Порядок работы с программой

Приложение предназначено для быстрого создания и подготовки к печати профессиональных ID-карт сотрудников компании Midea (MBY). Генерация выполняется за четыре последовательных шага.

  • Шаг 1. Загрузка макета: Кликните по кнопке «Выберите файл» в блоке «Фоновое изображение» и укажите шаблон карты. Макет мгновенно появится в зоне превью.
  • Шаг 2. Формат ФИО: В режиме «ФИО полностью» данные пишутся в одну строку через пробел. В режиме «Без отчества» графа отчества скрывается, а Фамилия и Имя автоматически сдвигаются ниже для сохранения баланса.
  • Шаг 3. Подразделение, ID и Фото: Выберите нужный «Отдел» из выпадающего списка и укажите цифровой ID. При необходимости активируйте чекбокс «Добавить фото сотрудника». Программа откроет всплывающий редактор, который позволит идеально кадрировать снимок под стандартные пропорции карты. При восстановлении карточки из истории специальный индикатор покажет имя ранее привязанного файла.
    В случае необходимости добавления или изменения какой-либо информации (например, в выпадающем списке "Отдел"), свяжитесь, пожалуйста с разработчиком приложения.
  • Шаг 4. Экспорт: При заполнении обязательных полей нижняя кнопка станет фиолетовой. Нажмите «Скачать ID-карту (PNG)», чтобы сохранить файл в печатном качестве 300 DPI со строгим автоматическим приведением регистра букв к правильному виду. Фотография экспортируется с тройной плотностью пикселей для идеальной четкости лица на печати.

Правила переноса в Microsoft Word (.DOCX)

Чтобы при выгрузке на печать из Word карточка сотрудника сохраняла свои точные физические размеры и не размывалась, используйте встроенные инструменты импорта:

  1. Откройте Word, перейдите во вкладку «Вставка» ➔ «Рисунки» ➔ «Это устройство» и укажите скачанную карту PNG.
  2. Кликните правой кнопкой мыши по изображению и выберите пункт «Размер и положение...».
  3. Во вкладке Размер убедитесь, что масштаб по ширине и высоте зафиксирован строго на 100%. Вшитые в код метаданные 300 DPI автоматически установят точные размеры карточки на листе: 5.4 см в ширину и 8.6 см в высоту (стандарт пластиковых карт ISO 7810 ID-1).
  4. Во вкладке Обтекание текстом задайте режим «Перед текстом» для свободного перемещения пропуска мышкой по странице.
